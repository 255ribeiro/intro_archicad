Roteiro 01 - Informações do projeto


UNIVERSIDADE SENAI CIMATEC
CURSO DE ARQUITETURA E URBANISMO







Docente: Prof: Msc. Fernando Ferraz Ribeiro Semestre: 2025.1


Objetivo da avaliação

  • Compreender funcionamento da ferramenta e sua interface.
  • Definir atributos de elementos iniciais para começar a modelagem BIM.
  • Utilizar conceitos BIM para modelo digital de construção.

Conteúdos Formativos

  • Apresentação do software e sua interface.
  • Definição de unidades do projeto;
  • Organização do projeto;
  • Definição dos atributos de elementos.

Introdução

Ao longo dessa disciplina será criado um modelo de Residência para que sejam praticados os comandos, ferramentas e utilidades do programa dentro do conceito BIM. Ao final de cada aula/roteiro o arquivo deverá ser salvo com o nome: Residência_Nome do aluno_Turma_Ano. O arquivo será utilizado como método avaliativo dos processos desenvolvidos ao longo desse roteiro. O objetivo principal dessa primeira aula é apresentar a interface do software Archicad com todas as abas e funcionalidades. A partir disso serão organizadas as informações do projeto no arquivo, considerando as boas práticas do desenvolvimento do mesmo. O template BR do Archicad será utilizado como base para todos os alunos, serão feitas algumas alterações ao longo das aulas para compreender os recursos disponíveis.

Descrições e Procedimentos

  1. Abra o Archicad, escolha criar um novo projeto.
  2. Prrencha seus dados em Arquivo -> Informações do projeto e preencha as informações conforme indicado abaixo:
  3. Nome do projetista (o seu nome)
  4. Nome do projeto (nome da disciplina e do módulo)
  5. Cliente (nome do professor)
  6. Verifique as configurações de unidades em: Opções -> Preferência do projeto -> Unidades de Trabalho.
  7. Em Opções> Preferência do projeto> Definições de localização simule a inserção de coordenadas geográficas. Veja também as opções:
    • Opções> Preferência do projeto> Definir Norte do Projeto
  8. Explore Janela -> Barra de Ferramentas -> Paletas e Janela -> Barra de Ferramentas -> Barra de Ferramentas
  9. Na Caixa de ferramentas explore as opções de desenho de:
  10. paredes
  11. pisos
  12. portas
  13. janelas
  14. Explore a navegação pelas vistas 2D e 3D.
  15. Ao desenhar pisos, vejas as configurações de níveis em:
  16. Mapa de projetos -> clicando o botão direito em Pisos -> Definições de piso
  17. Ou pelo atalho Ctrl + 7
  18. Salve o arquivo com o seguinte nome: Nome do aluno_aula_ano_mes_dia. Salvar como pln.
  19. Ao final da exploração, envie o arquivo pelo canvas.
